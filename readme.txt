== Description ==
Easily save and export information submitted by users through your website’s forms.
This plugin allows you to store and view information submitted by users through forms created with Jupiter X in the WordPress admin panel.
You can also export the information to a CSV file, which can be imported into Excel or other formats to process the data received through your website’s forms.

== Installation ==
**How to Install**

Installing the plugin is very easy:

1. Go to the WordPress admin of the website where you want to install it
2. Select the “Plugins -> Add New” option from the left sidebar menu
3. Search for ‘Save Forms for Jupiter X’
4. Click the “Install Now” button
5. Once installed, click the “Activate” button.

Once the plugin is installed, the option named Save & Export Forms will appear in the WordPress admin menu:
Clicking on this option will take you to the list of forms you have added.

**How to Use**

To add a form to save the information submitted by users, follow these steps:

1. Select the Jupiter X form where you want to save the data
2. You will see a new action called “Local Save”
3. Apply the action and save the page
